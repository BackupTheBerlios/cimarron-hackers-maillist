    con el señorito lenton hemos estado laburando estas 5 semanas para poner a
punto la segunda versión de cimarrón. ésta es una reescritura de cero, con un
ejemplo para que lo vayan viendo. warning: rompe compatibilidad para atrás, pero
por suerte no teníamos una base de usuarios muy grande (hola marce!).

    cosas que faltan: documentación (real soon now) y un grid que sirva de algo
(hay una, pero medio que apesta). cosas que sí andan que antes no: tenemos tests
que prueban que el bicho anda bien, búsqueda que funciona como debiera, y una
interfaz un poco mejor definida. obténganlón de:

https://developer.berlios.de/project/showfiles.php?group_id=2954
